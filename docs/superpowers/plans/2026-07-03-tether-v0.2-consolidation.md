# tether — Memory Consolidation Implementation Plan (Bet 2)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Keep the memory store from rotting — add temporal supersession (mark-invalid, never silently overwrite), recency-weighted recall, opt-in near-duplicate consolidation, and opt-in time-decay — with a **multi-writer-aware** schema so the future team tier is an ACL wrapper, not a data migration.

**Architecture:** Layers on top of Bet 1 (semantic search). Memories gain temporal columns (`valid_from` / `valid_to` / `superseded_by`) and an `author`; the "current" store is `valid_to IS NULL`. `remember` records authorship + validity, and (opt-in, using Bet 1's embeddings) can supersede a near-duplicate instead of fragmenting the store. `recall` excludes superseded rows and folds a **gentle** recency signal — and an opt-in exponential time-decay — into Bet 1's RRF fusion. Nothing is ever hard-deleted by consolidation; `forget` remains the only destructive path (the PII/compliance escape hatch). Safe defaults, powerful opt-ins.

**Tech Stack:** Python ≥3.10, stdlib `sqlite3` + `datetime`/`math`; `numpy` (already a Bet 1 `[semantic]` dep) for near-duplicate cosine. No new third-party dependency.

## Test Strategy

**Philosophy:** TDD per task. The default `pytest` run stays **hermetic** — no network, no model download — reusing Bet 1's deterministic `FakeEmbedder`. Time-dependent behavior (recency, decay) is tested by writing explicit ISO timestamps into rows via SQL, never by freezing the clock.

**Levels:**
- *Unit* — config resolvers; pure helpers (`_rrf_scores`, `_decay_factor`, `_age_days`); each `Store` method against an in-memory DB.
- *Integration* — `server._get_store()` wiring (consolidation defaults off); the existing MCP roundtrip is unaffected (still keyword-only, `TETHER_SEMANTIC=0`).

**Hermeticity controls:**
- `FakeEmbedder` (from Bet 1's test file) for supersession tests; `pytest.importorskip("numpy")` on the near-duplicate path.
- Timestamps set directly via `UPDATE memories SET updated_at=?` so recency/decay tests are deterministic and fast.
- Consolidation and decay are env-gated (`TETHER_CONSOLIDATE`, `TETHER_DECAY_HALF_LIFE_DAYS`); tests set them explicitly per-case.

**Deliberate non-goals (decided, not accidental):**
- **`forget` does not repair dangling `superseded_by` pointers.** If a user hard-deletes a row that had superseded others, the pointer dangles harmlessly (recall filters on `valid_to`, not on the pointer). Repairing it is not worth the complexity for a personal tool; revisit with the team tier.
- **No cross-author conflict policy.** The `author` column is *groundwork*; multi-writer dedup scoping / "who may invalidate whom" is the team-tier (ACL) concern, explicitly out of scope here.
- **No LLM-based importance scoring or summarize-then-drop.** Rejected in the research (expensive/drift-prone; lossy). Not built.
- **Latency of near-duplicate scan is not asserted in CI** (same rationale as Bet 1 — brute-force cosine over current same-type rows is <10 ms at target scale; validate manually if ever in doubt).

    except ValueError:
        return None
    return val if val > 0 else None

```python
    def _find_near_duplicate(self, type, emb):
        """Id of the most-similar CURRENT same-type memory whose cosine
        similarity to `emb` meets the dedup threshold, or None. Degrades to
        None (no consolidation) whenever semantic support is unavailable."""
        if emb is None or self._embedder is None:
            return None
        try:
            import numpy as np

            q = np.frombuffer(emb, dtype="<f4")
            rows = self._conn.execute(
                "SELECT id, embedding FROM memories "
                "WHERE type=? AND valid_to IS NULL AND embedding IS NOT NULL",
                (type,)).fetchall()
            best_id, best_sim = None, -1.0
            for mid, blob in rows:
                sim = float(np.frombuffer(blob, dtype="<f4") @ q)  # both unit-norm
                if sim > best_sim:
                    best_id, best_sim = mid, sim
            return best_id if best_sim >= self._dedup_threshold else None
        except Exception:
            return None
```

_RECENCY_WEIGHT = 0.25


def _rrf_scores(ranked_lists, k=60):
    """Reciprocal Rank Fusion as a {id: score} map (deterministic)."""
    scores = {}
    for lst in ranked_lists:
        for rank, mid in enumerate(lst):
            scores[mid] = scores.get(mid, 0.0) + 1.0 / (k + rank + 1)
    return scores


def _rrf_fuse(ranked_lists, k=60):
    return [mid for mid, _ in sorted(
        _rrf_scores(ranked_lists, k).items(), key=lambda kv: (-kv[1], kv[0]))]


def _age_days(iso: str, now_iso: str) -> float:
    a = datetime.fromisoformat(iso)
    b = datetime.fromisoformat(now_iso)
    return max(0.0, (b - a).total_seconds() / 86400.0)


def _decay_factor(age_days: float, half_life_days: float) -> float:
    return 0.5 ** (age_days / half_life_days)

    def recall(self, query, type=None, limit=20) -> list:
        if not query or not query.strip():
            return []
        fts_ids = self._fts_ids(query, type)
        vec_ids = self._vector_ids(query, type)
        lists = [fts_ids] + ([vec_ids] if vec_ids else [])
        scores = _rrf_scores(lists)
        if not scores:
            return []
        # gentle recency signal: breaks ties, never overrides a strong match
        recency = _rrf_scores([self._recency_order(list(scores))])
        for mid, s in recency.items():
            scores[mid] += _RECENCY_WEIGHT * s
        # optional exponential time-decay
        if self._decay_half_life_days:
            now = _now()
            updated = self._updated_at_of(list(scores))
            for mid in list(scores):
                scores[mid] *= _decay_factor(
                    _age_days(updated[mid], now), self._decay_half_life_days)
        order = [mid for mid, _ in sorted(
            scores.items(), key=lambda kv: (-kv[1], kv[0]))][:limit]
        return self._hydrate(order)

    def _recency_order(self, ids):
        if not ids:
            return []
        ph = ",".join("?" for _ in ids)
        return [r[0] for r in self._conn.execute(
            f"SELECT id FROM memories WHERE id IN ({ph}) "
            f"ORDER BY updated_at DESC, id DESC", ids).fetchall()]

        if not ids:

tether keeps a superseded fact rather than overwriting it: when a memory is
